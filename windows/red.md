# Comandos de red en Windows

- `ipconfig` → Muestra configuración de red.
- `ping <host>` → Prueba conexión.
- `tracert <host>` → Muestra la ruta de red.
- `netstat -an` → Muestra puertos abiertos.
- `nslookup <dominio>` → Consulta DNS.
- `arp -a` → Muestra la tabla ARP.
- `route print` → Muestra rutas de red.
- `net use` → Mapea unidades de red.
- `curl <url>` → Descarga contenido de una URL.
- `ftp` → Conexión FTP.
