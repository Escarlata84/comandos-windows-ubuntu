# Programas, procesos y sistema

- `tasklist` → Lista procesos en ejecución.
- `taskkill /PID <id> /F` → Finaliza un proceso.
- `shutdown /s /t 0` → Apaga el equipo.
- `shutdown /r /t 0` → Reinicia el equipo.
- `chkdsk` → Revisa el disco.
- `sfc /scannow` → Repara archivos del sistema.
- `winget search <programa>` → Busca programas (Windows moderno).
- `winget install <programa>` → Instala programas.
- `wmic cpu get name` → Muestra info del CPU.
- `powercfg /batteryreport` → Reporte de batería (portátiles).
