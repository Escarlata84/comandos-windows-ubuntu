# Archivos y carpetas en Windows

- `dir` → Lista archivos y carpetas.
- `cd` → Cambia de directorio.
- `mkdir <carpeta>` → Crea una carpeta.
- `rmdir <carpeta>` → Elimina una carpeta.
- `copy <origen> <destino>` → Copia archivos.
- `xcopy <origen> <destino>` → Copia carpetas y archivos.
- `move <origen> <destino>` → Mueve archivos o carpetas.
- `del <archivo>` → Elimina archivos.
- `type <archivo>` → Muestra el contenido de un archivo.
- `attrib` → Muestra o cambia atributos de archivos.
