# Comandos básicos de Windows

- `cls` → Limpia la pantalla de la consola.
- `echo` → Muestra un mensaje en pantalla.
- `ver` → Muestra la versión de Windows.
- `help` → Muestra ayuda general.
- `exit` → Cierra la consola.
- `time` → Muestra o cambia la hora.
- `date` → Muestra o cambia la fecha.
- `hostname` → Muestra el nombre del equipo.
- `whoami` → Muestra el usuario actual.
- `systeminfo` → Muestra información del sistema.
