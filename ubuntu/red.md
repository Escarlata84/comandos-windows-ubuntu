# Comandos de red

- `ping <host>` → Envía paquetes ICMP para probar conexión.
- `ifconfig` → Muestra interfaces de red (alternativa: `ip a`).
- `ip a` → Muestra interfaces de red.
- `curl <url>` → Descarga contenido de una URL.
- `wget <url>` → Descarga archivos desde internet.
- `netstat -tulnp` → Muestra puertos abiertos y servicios.
- `ssh <usuario>@<host>` → Conexión remota por SSH.
- `scp <archivo> <usuario>@<host>:<ruta>` → Copia archivos vía SSH.
- `traceroute <host>` → Muestra ruta hasta un host.
- `nslookup <host>` → Consulta DNS.
