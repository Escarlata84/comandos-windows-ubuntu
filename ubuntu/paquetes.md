# Comandos de paquetes y software

- `sudo apt update` → Actualiza la lista de paquetes.
- `sudo apt upgrade` → Actualiza los paquetes instalados.
- `sudo apt install <paquete>` → Instala un paquete.
- `sudo apt remove <paquete>` → Elimina un paquete.
- `sudo apt autoremove` → Elimina paquetes innecesarios.
- `dpkg -l` → Lista paquetes instalados.
- `dpkg -i <archivo.deb>` → Instala paquete .deb.
- `snap install <paquete>` → Instala paquete snap.
- `snap remove <paquete>` → Elimina paquete snap.
- `flatpak install <repositorio> <paquete>` → Instala paquete Flatpak.
