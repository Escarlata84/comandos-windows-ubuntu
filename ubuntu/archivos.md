# Comandos de archivos y carpetas

- `mkdir <nombre>` → Crea una carpeta.
- `rmdir <carpeta>` → Elimina una carpeta vacía.
- `touch <archivo>` → Crea un archivo vacío.
- `rm <archivo>` → Elimina un archivo.
- `cp <origen> <destino>` → Copia archivos.
- `mv <origen> <destino>` → Mueve o renombra archivos.
- `cat <archivo>` → Muestra el contenido de un archivo.
- `less <archivo>` → Muestra contenido de archivo página por página.
- `head <archivo>` → Muestra las primeras líneas de un archivo.
- `tail <archivo>` → Muestra las últimas líneas de un archivo.
