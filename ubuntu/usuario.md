# Comandos de usuarios

- `whoami` → Muestra el usuario actual.
- `id` → Muestra información del usuario y grupos.
- `adduser <usuario>` → Crea un nuevo usuario.
- `passwd <usuario>` → Cambia la contraseña de un usuario.
- `groups` → Muestra los grupos a los que pertenece el usuario.
- `sudo <comando>` → Ejecuta un comando como superusuario.
- `su - <usuario>` → Cambia de usuario.
- `who` → Lista usuarios conectados.
- `w` → Muestra usuarios conectados y actividad.
- `last` → Muestra últimos inicios de sesión.
